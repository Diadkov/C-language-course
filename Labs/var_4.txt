Запишіть у файл результат програми, вказуючи кількість входжень для кожного з них.(якщо це наприкладінвертування бітів має
 бути масив інвертованих значень записані у файл)

Зчитування з файлу:
Зчитайте дані з файлу і виведіть їх на екран.
Завдання (допуск номер 3)

Визначте структуру, яка міститиме поля, наприклад, id та value.
Створення масиву структур:

Створіть масив або вектор структур, заповнивши його даними.
Запис у файл:

Відкрийте файл для запису (наприклад, results.txt).
Запишіть у файл дані з масиву структур, розділяючи поля пробілами або комами.
Зчитування з файлу:

Відкрийте файл для читання.
Зчитайте дані та відтворіть структуру в пам’яті за допомогою динамічного масиву, потім виведіть їх на екран.
