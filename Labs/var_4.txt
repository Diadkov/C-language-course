Завдання 1.13: Гістограма
Створити гістограму за доп програми 1.13 вертикальної:

Зберіть дані, для яких потрібно створити гістограму.
Підрахуйте, скільки разів кожне значення зустрічається у ваших даних.
Запис гістограми у файл:

Відкрийте файл для запису (наприклад, histogram.txt).
Запишіть у файл значення гістограми, вказуючи кількість входжень для кожного з них.
Зчитування з файлу:

Відкрийте файл, в якому зберігалася гістограма.
Зчитайте дані з файлу і виведіть їх на екран.

Визначте структуру, яка міститиме поля, наприклад, id та value.
Створення масиву структур:

Створіть масив або вектор структур, заповнивши його даними.
Запис у файл:

Відкрийте файл для запису (наприклад, results.txt).
Запишіть у файл дані з масиву структур, розділяючи поля пробілами або комами.
Зчитування з файлу:

Відкрийте файл для читання.
Зчитайте дані та відтворіть структуру в пам’яті, потім виведіть їх на екран.
